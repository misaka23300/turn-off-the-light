## 使用功能：
    当教室无人上课，或者到下课时间后，忘记关灯。通过传感器检测是否有人存在，实现自动关灯，节约能源。

## 关闭逻辑：
    每天的中午12点和晚上8点是下课的时间，到点后检测是否有人，无人就关灯。
    如果麦克风检测不到声音，并且人体传感器也检测不到人体，开始执行关灯。

## 可靠性
    人体检测模块无法覆盖全教室。
    如果有人上自习可能误关灯。
    不推荐实行第二种检测方法。

## 硬件结构
- 麦克风 -> 响度
- 舵机 -> 开关灯
- 人体传感器 -> 是否有人

引脚
- 麦克风 3.3V
    - SD -> GPIO10
    - SCK -> GPIO9
    - WS -> GPIO8
- 舵机 5V
    - pwm -> GPIO6
- 24G检测模块 5V /3.3V
    - GPIO7

## 软件结构
- 每1分钟检测麦克风响度。
- 如果没有响度，检测是否有人。
- 每到12点和8点，检测是否有人，实现关灯。
